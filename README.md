# Auto-checker for Kemono.party/Kemono.su (Windows only)

The program will periodically check **kemono.party** for new posts for whatever creators you want.

*Services Supported:*
- Patreon
- Fanbox
- Boosty
- DLsite
- GumRoad
- SubscribeStar

*Features:*
- Persistent storage
- Thorough logging
- Tray icon interaction
- Notification with direct link
